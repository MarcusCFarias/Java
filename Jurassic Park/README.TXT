  DESCRIÇÃO:
 - O úsuario tem a possibilidade de cadastrar dinossauros, caso positivo pode ainda escolher quantos quer cadastrar.
 - 3 dinossauros são sempres cadastrados automaticamente.
 - O usuário informa o tamamnho do parque (que é montado como uma matriz quadrada) a partir do comprimento da lateral. É verificado se existe espaço o suficiente para que
 cada dinossauro use uma posição da matriz, caso negativo é pedido que seja informado um novo tamanho para o parque.
 - Sempre que o usuário cadastra um dinossauro ele deve também informar a sua posição no parque indicando a coluna e linha que este se encontra.
 - Com o cadastro completo o úsuario podebuscar as informações que deseja a partir dos relatórios disponíveis.
